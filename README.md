# NorthstarCN_version_switch
A batch file for quickly switching between versions 1.17.7 and 1.18.0 locally. (For personal use)
***
##File Structure
SteamLibrary\steamapps\common\Titanfall2\
- _ns_versions\
  - 1.17.7\
    - Northstar.dll
    - NorthstarLauncher.exe
    - bin\x64_retail\wsock32.dll
    - mods\
      - Northstar.Client
      - Northstar.Custom
      - Northstar.CustomServers
  - 1.18.0\
    - Northstar.dll
    - NorthstarLauncher.exe
    - bin\x64_retail\wsock32.dll
    - mods\
      - GtJt.HUDRevamp+KS8
      - Northstar.Client
      - Northstar.Custom
      - Northstar.CustomServers
- （GAME ROUTE）
